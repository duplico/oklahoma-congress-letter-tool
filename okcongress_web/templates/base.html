About

Purpose

The purpose of this site is to provide a quick tool to generage letters to Oklahoma legislatures, based on individual selection, or committee membership. It will help you generate (a) a list of phone numbers to call and a script to read based on your goal and specific legislation, and/or (b) a stack of letters and envelopes to print, sign, and mail to those legislators.

Data source

The source of legislator office numbers, phone numbers, and committee membership data is from a script that scrapes the official sites. The house data is decent, but the senate site is a mess. Because Oklahoma does not provide legislative data in machine readable format, web scraping is unfortunately the best alternative. Please report any errors.

Author

This is a quick weekend project of George Louthan. Views are my own, etc. The code is available on my bitbucket. This is a work in progress. Contributors are welcome, whether an activist looking for feature requests, or a developer looking to pitch in.
